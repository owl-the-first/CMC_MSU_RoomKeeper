# CMC MSU RoomKeeper Bot

Telegram-бот для поиска и бронирования свободных аудиторий на факультете ВМК МГУ.

## Возможности бота
    - загрузка расписания с сайта ВМК
    - парсинг занятых аудиторий
    - поиск свободных аудиторий по дате и времени
    - создание заявки на бронирование
    - просмотр своих бронирований

## Источник расписания

Официальная страница расписания ВМК МГУ:
https://cs.msu.ru/studies/schedule
