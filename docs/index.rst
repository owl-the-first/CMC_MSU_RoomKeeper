CMC MSU RoomKeeper
==================

CMC MSU RoomKeeper — это Python-проект для поиска и бронирования
свободных аудиторий на факультете ВМК МГУ.

Проект умеет скачивать PDF-файлы расписания, извлекать из них занятость
аудиторий, сохранять данные в базу SQLite и предоставлять пользователю
Telegram-бота для поиска свободных аудиторий и создания заявок на бронь.
