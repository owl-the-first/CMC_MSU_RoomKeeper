Использование
=============

Установка
---------

Создать виртуальное окружение, активировать его и установить проект с
зависимостями для разработки:

.. code-block:: bash

   python -m venv .venv
   source .venv/bin/activate
   python -m pip install ".[dev]"

Для editable-установки (``pip install -e ".[dev]"``) на Python 3.12 может
потребоваться дополнительная настройка; надёжнее использовать обычную
установку командой выше.

После установки появляются консольные команды:

.. code-block:: bash

   roomkeeper-init-db
   roomkeeper-update-schedule
   roomkeeper-import-schedule
   roomkeeper-db-stats
   roomkeeper-bot

Настройка окружения
-------------------

Скопировать пример и указать значения:

.. code-block:: bash

   cp .env.example .env

Основные переменные:

``TELEGRAM_BOT_TOKEN``
   Токен Telegram-бота.

``ROOMKEEPER_DATABASE_URL``
   URL базы данных (по умолчанию ``sqlite:///data/roomkeeper.db``).

``ROOMKEEPER_DEFAULT_LOCALE``
   Язык интерфейса по умолчанию: ``ru`` или ``en``.

``TELEGRAM_ADMIN_IDS``
   Числовые Telegram ID администраторов через запятую. Администраторы
   получают уведомления о новых заявках и могут их одобрять или отклонять.

Инициализация базы данных
-------------------------

Команда создаёт таблицы базы данных:

.. code-block:: bash

   roomkeeper-init-db

По умолчанию используется локальная SQLite-база данных.

Обновление расписания
---------------------

Команда скачивает PDF-файлы расписания, извлекает из них текст и сохраняет
результат парсинга в JSON-файл:

.. code-block:: bash

   roomkeeper-update-schedule

После выполнения команды появляются локальные данные в каталогах ``data/raw``
и ``data/parsed``.

Импорт расписания в базу данных
-------------------------------

После парсинга расписание можно импортировать в базу:

.. code-block:: bash

   roomkeeper-import-schedule

Просмотр статистики
-------------------

Для быстрой проверки содержимого базы используется команда:

.. code-block:: bash

   roomkeeper-db-stats

Telegram-бот
------------

Для запуска бота нужна переменная окружения с токеном Telegram-бота.
Пример настройки окружения хранится в файле ``.env.example``.

Запуск бота:

.. code-block:: bash

   roomkeeper-bot

Основные команды бота
---------------------

``/start``
   Показывает приветственное сообщение.

``/help``
   Показывает список доступных команд.

``/free``
   Ищет свободные аудитории на заданный день и интервал времени.

``/book``
   Создаёт заявку на бронирование аудитории. Администраторы получают
   уведомление о новой заявке.

``/my_bookings``
   Показывает заявки текущего пользователя.

``/cancel_booking``
   Отменяет заявку по номеру. Короткий вариант: ``/cancel``.

``/lang``
   Переключает язык интерфейса: ``/lang ru`` или ``/lang en``.

Команды администратора
----------------------

Доступны только пользователям из ``TELEGRAM_ADMIN_IDS``.

``/admin``
   Справка по админ-командам.

``/pending``
   Показывает заявки, ожидающие решения.

``/approve НОМЕР``
   Подтверждает заявку.

``/reject НОМЕР``
   Отклоняет заявку.

При уведомлении о новой заявке администратор может нажать кнопки
«Одобрить» или «Отклонить» прямо в чате.
